# Typing Speed Test

An interactive web application to test and improve your typing speed and accuracy. Developed by **Developer Gowtham**, this project includes personalized typing certificates, multiple difficulty levels, and detailed performance statistics.

## Features

- **Interactive Typing Test**: Select difficulty levels (Easy, Medium, Hard) and test your typing skills.
- **Performance Feedback**: Displays total words attempted, correct words, mistakes, and accuracy.
- **Personalized Certificate**: Download a certificate upon completion of the test.
- **Responsive Design**: Fully optimized for desktop and mobile devices.
- **Contact Form**: Integrated contact form for user feedback and queries.

## How to Use

1. **Enter Your Name**: Start the test by entering your name.
2. **Select Difficulty**: Choose a difficulty level (Easy, Medium, or Hard).
3. **Start Typing**: Press "Start Typing" and type the words displayed on the screen.
4. **View Results**: Review your typing performance, including accuracy and mistakes.
5. **Download Certificate**: Celebrate your achievement by downloading a personalized typing certificate.
6. **Contact**: Use the contact form to share your feedback or ask questions.
